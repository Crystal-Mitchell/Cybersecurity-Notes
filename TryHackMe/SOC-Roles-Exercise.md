# SOC Roles Exercise

## Objective

Match cybersecurity tasks to the appropriate security role within a Security Operations Center (SOC).

## Key Roles Reviewed

### SOC L1 Analyst
- Monitors alerts
- Performs initial triage
- Escalates incidents

### SOC L2 Analyst
- Performs deeper investigations
- Analyzes escalated incidents
- Determines scope and impact

### CERT Lead
- Coordinates incident response
- Handles major security incidents

### GRC Auditor
- Assists with compliance and audits
- Reviews regulatory requirements

### Penetration Tester
- Identifies vulnerabilities
- Tests security controls

### SOC Engineer
- Maintains SIEM infrastructure
- Troubleshoots platform issues

### Threat Researcher
- Studies threat actors
- Analyzes tactics, techniques, and procedures (TTPs)

## Challenge Encountered

I initially confused the responsibilities of SOC L1 and SOC L2 analysts and answered several questions incorrectly.

## Key Takeaway

SOC L1 analysts perform initial alert monitoring and triage.

SOC L2 analysts perform deeper investigations and analysis of escalated incidents.

This exercise reinforced the importance of understanding not only security concepts but also the responsibilities of different cybersecurity roles within a SOC.
